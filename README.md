# Drawing3dSphere
Drawing a 3D Sphere in C++

```cpp
int iSides = pExplosionEntityData->m_BlastRadius * 8;
if ( Settings->m_bGrenadeSphere )
    DrawSphere( position, transform, pExplosionEntityData->m_BlastRadius, iSides, dwColor );
```
